# CMS Editor Guide (Webmaster)

## 1) Access

- URL: `https://cms.poweramazonica.com/admin`
- Use your editor credentials.
- If login fails, contact Admin (do not create duplicate users).

## 2) What You Can Edit

- Home singleton:
  - Hero text and CTA labels/links.
  - About section.
  - Location and advantages text.
  - Final CTA and footer contact fields.
- Collections:
  - `slides` for hero/section slider.
  - `project_features` cards.
  - `payment_options`.
  - `faqs`.

## 3) Standard Publishing Flow

1. Open the section to edit.
2. Update text/images/order.
3. Save draft.
4. Review content carefully:
   - spelling
   - phone/mail/WhatsApp links
   - image quality and crop
5. Publish.
6. Verify the public page in browser incognito.

## 4) Content Rules

- Do not publish pricing or guarantees unless approved by PM/legal.
- Keep language compliant:
  - use "alta proyeccion" style wording.
  - avoid absolute promises.
- Keep CTA links valid:
  - WhatsApp `https://wa.me/51990814630?...`
  - Phone `tel:+51990814630`
  - Email `mailto:orbezog@hotmail.com`

## 5) Image Rules

- Use high quality images with consistent style.
- Prefer WebP/JPEG optimized for web.
- Avoid heavy files when possible.
- Keep visual coherence with premium real-estate style.

## 6) Quick QA Before Publish

- Home loads correctly on mobile and desktop.
- Main CTA buttons open correct apps/links.
- Slider (when enabled) navigates and images are visible.
- FAQ content is readable and accurate.
- Footer contact data is correct.

## 7) If Something Goes Wrong

- Stop publishing further edits.
- Notify Admin/DevOps with:
  - edited collection/item
  - approximate time
  - screenshot
- Admin can restore previous revision or run rollback procedure.
